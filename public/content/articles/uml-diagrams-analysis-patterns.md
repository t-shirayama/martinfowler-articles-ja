# UML Diagrams for Analysis Patterns

## 要約

この短い補足ページは、`Analysis Patterns` の各章に対応する UML 図を整備しようとした経緯を説明しています。
Martin Fowler 氏自身は十分な時間を取れず、いくつかの章については協力者が図を提供しています。

## 読むときの観点

- `Analysis Patterns` の補足資料として位置づける。
- 図は本文の公式な置き換えではなく、章ごとの理解を助ける追加資料として読む。
- 協力者によって章ごとの図が追加されていることを確認する。

## 原文の翻訳

これは、ほとんど謝罪のようなものです。
私は、自分の analysis patterns の UML 版を作るつもりでいました。
しかし、それはかなりの作業であり、私はほかのプロジェクトで忙しすぎました。

第2章は、Michael Feathers の親切な取り組みによるものです。
Igor Janicijevic も加わり、第3章、第4章、第5章の図を追加してくれました。
Jennifer Hansen は第6章の図を提供してくれました。
Pierre de Swert は第8章の図を提供してくれました。

さまざまな理由で完全には整理しきれなかったものの、試みてくれたほかの人たちにも感謝したいと思います。
これを読んでいる人の中で、どこかの章に取り組んでみたい人がいれば、助けてもらえるとありがたいです。
私は喜んで図を確認し、このページに掲載し、あなたの貢献としてクレジットを示します。
