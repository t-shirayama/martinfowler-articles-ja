# Communal Dashboard

## 要約

Communal Dashboard は、組織内の半公共的な場所に置かれたディスプレイで、進行中の事業や活動の状態を共有するためのダッシュボードです。
個人が詳細に分析して意思決定するためというより、周囲の人たちが自然に状況を知り、仕事の背景や文脈を得るために使われます。

この種の表示は、ライブデータにつながった Information Radiator の一種です。
ただし、画面を置くだけで人が関心を持つとは限らず、表示内容を会話につなげる人間的な働きかけも重要になります。

## 読むときの観点

- ダッシュボードを個人向け分析ツールではなく、共有された状況認識の道具として見る。
- 詳細さや対話性よりも、周囲の人の注意を引き、会話の入口を作る役割に注目する。
- 見せ方の工夫が、データとの感情的なつながりを作る点を見る。
- 画面表示だけでなく、リーダーや周囲の人がどう会話へつなげるかを考える。

## 原文の翻訳

データ分析や可視化への関心が高まるにつれて、組織内を流れているデータから洞察を得られるようにする、興味深い可視化への取り組みが増えています。こうしたダッシュボードの多くは個人利用を想定していますが、より共同的な目的で使おうとする傾向も強まっています。

考え方は、組織内の半公共的な場所に表示用モニターを置くことです。そうすると、多くの人が Communal Dashboard の情報を見ることができます。これは、人々の意思決定を助けるというより、何が起きているかを人々に知らせ、日々の仕事のための背景情報や文脈を提供するためのものです。ある人は、事業がどのように動いているかに対する感情的な関わりを作るものだと表現していました。アジャイルソフトウェアのコミュニティにいる人なら、自然にこれを Information Radiator の一形態と見るでしょう。

私は Communal Dashboard を、Information Radiator の下位型だと見ています。つまり、**ライブのデータフィードにつながっていて動的である**という特徴を持つものです。紙のポスターのような静的な掲示物も Information Radiator の例ですが、Communal Dashboard ではありません。ただし、そうした掲示物も非常に効果的であることはよくあります。

この例のひとつが、北米でDVDレンタルの自動販売機を運営する Redbox です。同社のダッシュボードには、これまでの総レンタル数の実行中カウント、最近よくレンタルされた映画の一覧、1秒あたりのレンタル数を示すスピードメーターなどのデータが表示されます。情報はオフィス内のさまざまなディスプレイに表示されます。安価な薄型モニターが普及したことで、こうした取り組みは以前よりずっと低コストでできるようになりました。

想像力のある表示は、人の注意を引く助けになります。Redbox の表示のひとつには、過去7日間に新しい Redbox の機械が設置された場所を地図上の押しピンで示すものがありました。日が進み、週が進むにつれて、新しい Redbox の機械がどこに現れているかを見ることができます。

別の会社は、家や車に似た、見た目がそれぞれ異なる製品を販売しています。その会社では、製品が売れたときにその写真を、Apple の app wall に着想を得た、テトリス風に落ちてくるアイコン表示として見せていました。

通常、この用途には、フルスクリーンモードのWebブラウザがよい選択肢になります。JavaScript と HTML5 で動的な可視化を生成できるライブラリも、ますます利用しやすくなっています。

表示媒体にも、より想像力を働かせることができます。Redbox では、その日のレンタル数の実行中カウンターを、メインロビーの外の床に投影しています。

こうした可視化は粗いものです。多くの詳細を見せることはできず、対話的でもありません。それでも、何が起きているかへの認識を生み、オフィスのスタッフと、普段は離れたところにある日々の活動とのつながりを作れます。訪問してくるビジネスパートナーも、こうした表示の対象になります。このような画面上の情報は、取引を円滑にする会話の自然な入口になります。

こうしたダッシュボードがどれほどの影響を生むのかは、はっきりしないかもしれません。この種のダッシュボードをよく使っているある人は、人々がそれほど関心を示しているようには見えず、コーヒーを飲みながら新しい会話が始まるわけでもなかった、と述べていました。その人は、人々をこの情報に関わらせるには、表示されている内容に基づいてリーダーが質問を投げかけるなど、**人間的な働きかけ**が必要になることが多いと助言しています。
