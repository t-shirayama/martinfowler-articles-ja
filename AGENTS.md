# AGENTS.md

このリポジトリは、martinfowler.com に掲載された記事の非公式日本語訳を管理するためのものです。
作業するときは、公式プロジェクトと誤認されないこと、原文への敬意を保つことを最優先にしてください。

## 基本方針

- Martin Fowler 氏、martinfowler.com、Thoughtworks の公式翻訳であるような表現をしない。
- 元サイトのデザイン、ロゴ、配色、レイアウトをコピーしない。
- 翻訳記事には、原文タイトル、原文URL、著者名、翻訳日、非公式であることを必ず入れる。
- 原文の意味を変えず、日本語として読みやすい表現にする。
- 不確かな訳語は無理に確定せず、必要なら訳注や TODO として残す。

## Markdown の書き方

- 文章は日本語で書く。
- 見出しは短く、読みやすくする。
- コード、ファイル名、クラス名、メソッド名、英語の専門用語は必要に応じて原語を残す。
- 原文へのリンクは各記事の冒頭に置く。
- 公式サイトと混同されるような装飾やコピー文は避ける。

## 記事冒頭の必須情報

```markdown
> これは非公式の日本語訳です。Martin Fowler 氏、martinfowler.com、Thoughtworks の公式翻訳ではありません。

- 原文: [Original Title](https://martinfowler.com/...)
- 著者: Author Name
- 翻訳日: YYYY-MM-DD
```

## 作業時の注意

- 翻訳対象を追加するときは、まず `README.md` の記事候補一覧に載せる。
- 翻訳本文を追加するときは、原文へのリンクを確認する。
- 大きな構成変更をする前に、変更案を短く説明する。
- 著作権や出典表記に関わる判断は慎重に扱う。
